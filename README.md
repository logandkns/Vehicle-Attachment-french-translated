# Inferno Collection: Vehicle Attachment

__Public Beta Version 1.5__

A lightweight vehicle attachment/tow script for FiveM. Attach vehicles to other vehicles and adjust the positioning with easy to use keybinds. Multiple attachments to the same vehicles (such as a flatbed trailer) are also possible.

This resource requires at least OneSync \"legacy\". Use [Public Beta Version 1.3](https://github.com/inferno-collection/Vehicle-Attachment/releases/tag/v1.3-beta) if you do not want to use OneSync.

<a href="https://www.buymeacoffee.com/inferno_collection" target="_blank"><img width="245" height="45" alt="fire_truk" src="https://inferno-collection.com/img/fire_truk.png" /></a>

Presently, the following can be customized:
- Toggleable line that is drawn to show you what vehicle you are interacting with.
- Toggleable chat message instructions on how to move vehicles around.
- Attachment black/whitelist.
    - Toggleable between the two.
    - Toggleable option to have a complete whitelist, allowing control of which vehicles can and cannot be towed.
- Max number of vehicles attached to one tow vehicle
- Max distance from tow vehicles other vehicles can be attached.
- Control over how fast/slow attachment positioning is.
- Option to drive vehicles onto towing vehicle to confirm position (`/attach driveon`)

Usage:
- Go up to a vehicle and type `/attach` or `/attach driveon`
- Select the tow vehicle, and then vehicle to be towed with the `Enter` key
- Follow on screen instructions to position the vehicle
- Repeat for as many vehicles as you would like to attach
- When ready to detach, type `/detach` near a tow vehicle and vehicles will detach in the reverse order they were attached

# Inferno Collection: Vehicle Attachment

__Public Beta Version 1.5__

Un script léger de fixation/remorquage de véhicule pour FiveM. Fixez des véhicules à d'autres véhicules et ajustez leur position à l'aide de raccourcis clavier faciles à utiliser. Il est également possible de fixer plusieurs éléments au même véhicule (comme une remorque à plateau).

Cette ressource nécessite au moins OneSync "legacy". Cette version française ne bénéficie pas de Rétro-Compatibilité, OneSync "legacy" devient donc obligatoire si vous souhaitez l'avoir en français

<a href="https://www.buymeacoffee.com/inferno_collection" target="_blank"><img width="245" height="45" alt="fire_truk" src="https://inferno-collection.com/img/fire_truk.png" /></a>

Actuellement, les éléments suivants peuvent être personnalisés :
- Ligne activable/désactivable qui s'affiche pour vous indiquer avec quel véhicule vous interagissez.
- Messages de chat activables/désactivables contenant des instructions sur la manière de déplacer les véhicules.
- Black/Whitelist des attachments.
- Activable/désactivable pour les deux.
- Option activable/désactivable pour avoir une whitelist complète, permettant de contrôler quels véhicules peuvent et ne peuvent pas être remorqués.
- Nombre maximum de véhicules attelés à un véhicule de remorquage
- Distance maximale à laquelle les autres véhicules peuvent être attachés aux véhicules de remorquage.
- Contrôle de la vitesse/lenteur du positionnement de l'attache.
- Option permettant de conduire les véhicules sur le véhicule de remorquage pour confirmer la position (`/attacher driveon`)

- Approchez-vous d'un véhicule et tapez `/attacher » ou « /attacher driveon`.
- Sélectionnez le véhicule tracteur, puis le véhicule à tracter à l'aide de la touche « Entrée ».
- Suivez les instructions à l'écran pour positionner le véhicule.
- Répétez l'opération pour autant de véhicules que vous souhaitez atteler.
- Lorsque vous êtes prêt à dételer, tapez « /detacher » à proximité d'un véhicule tracteur et les véhicules se dételleront dans l'ordre inverse de celui dans lequel ils ont été attelés.
